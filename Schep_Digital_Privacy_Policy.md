# Privacy Policy for Schep_Digital Pinterest App

**Last Updated:** June 16, 2026

This Privacy Policy describes how the Schep Digital Pinterest App ("the App") collects, uses, and protects your information.

## 1. Information We Collect
**We do not collect, store, or process any personal data.** 
The App operates strictly as a one-way content management automation tool. Its sole purpose is to publish images and product links from our own secure servers directly to our own Pinterest boards. 

## 2. Pinterest User Data
The App does not request, access, or store any personal information from Pinterest users. We do not read user profiles, track user behavior, or collect analytics data through the Pinterest API.

## 3. How We Use Information
Because we collect no personal data, no data is used for marketing, tracking, or profiling. The App only uses our own developer API credentials to authenticate and post our own content.

## 4. Third-Party Sharing
We do not share any data with third parties, as no user data is ever collected or stored by the App.

## 5. Security
All authentication with the Pinterest API is handled securely via standard OAuth 2.0 protocols. API keys and secrets are stored in secure environment variables and are never exposed publicly.

## 6. Contact Us
If you have any questions or concerns about this Privacy Policy, please contact the developer via the associated GitHub repository or Gumroad store contact forms.
