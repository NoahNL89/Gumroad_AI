# Codex Image Library

Each numbered folder matches the live Gumroad catalog order.

## File roles

- `key-art.jpg` — square master image for the Gumroad product page, PDF cover, and promotional crops.
- `interior-triptych.jpg` — the three coordinated PDF illustrations in one source sheet.
- `interior-01-*.jpg` — concept or opening-section illustration.
- `interior-02-*.jpg` — process or workflow illustration.
- `interior-03-*.jpg` — outcome, advanced technique, or closing-section illustration.

All generated artwork is intentionally text-free so headings and captions can be
typeset cleanly in the PDF source without image-generation spelling errors.
